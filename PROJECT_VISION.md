## 🎯 核心诉求

**我们的核心诉求是**：前端表现必须要极其优秀，视觉设计、UI、动画转场、画风和美术必须具备极高的高级感和设计感，运行丝滑不卡，且具备极强的可玩性和多人竞技博弈深度。

## 🎨 第一部分：游戏核心愿景与美术审美要求

我们要做的不是一款传统的网页游戏，而是一件**极具视觉冲击力和沉浸感的交互艺术品**。

### 画风与美术
我们需要顶级的视觉审美，不管最终采用哪种美术表现形式，都必须呈现出极高的艺术质感。
- 光影、材质、环境氛围（如四季流转、自然天气系统）必须极其细腻生动
- 有高级的艺术表达

### 动画与转场
要求有着极致的动画细节和丝滑无缝的转场体验。
- 从单机切入联机、从农场进入战斗，所有的界面交互和状态切换都要连贯自然
- 拥有极佳的动态反馈，彻底摒弃生硬的画面切换

### UI 设计
采用极简、具备现代审美和极强设计感的界面语言：
- 排版精良
- 交互优雅
- 不落俗套

**当前美学方向**: 温暖奇幻手账风（羊皮纸背景 + 金色边框 + 衬线字体 + 四角装饰）

## 🎮 第二部分：游戏核心玩法与机制设定

这是一款将农场经营、生物养成、卡牌构筑以及大地图地缘博弈深度融合的游戏。

### 五大玩法支柱

#### 1. 活地图经营
- 土地有四季轮转、肥力与天气系统
- 农作物不仅仅是交易产物，更是制作材料和合成卡牌的核心
- 经营与战斗体系完全绑定

#### 2. 灵兽捕获与进化
- 野外分布着**200+ 种大地灵兽**
- 它们不仅用于战斗，更是农场的核心劳动力（如水系灌溉、风系驱动风车）
- 同一只灵兽可以由玩家培养进化出完全不同的方向

#### 3. 实物卡牌构筑
- 卡牌不是凭空随机掉落，而是**由玩家亲手打造**
- 土地产出材料 → 工坊锻造出卡牌 → 组成卡组带入地下城或 PvP
- 农场经营得越强，卡牌的上限越高

**当前实现**: 炼金工坊系统（6 条隐藏配方探索）

#### 4. 生态链与科技树
- 土地上有真实的食物链系统，比如过度捕猎某种灵兽会导致害虫爆发破坏农田
- 玩家可以选择农业、军事或魔法等不同的科技树发展
- 条条大路皆可获胜

#### 5. 大陆政治层与季节防御
- 全服共享一张大地图
- 联机时，邻近玩家的土地政策会真实影响你的农田气候（比如邻居大量砍伐导致你这边干旱）
- 每个季节末尾会有虚空潮汐入侵，玩家需要利用灵兽、卡组和防御设施抵御
- 甚至需要邻国互相支援

## ⏱️ 第三部分：游戏循环与联机架构设计

### 三层游戏节奏循环

#### 每日循环（短节奏）
- 每天分配体力进行耕种、探索、锻造与 NPC 互动
- 随时存档，体验完整的单机经营

#### 季节循环（中节奏）
全服联机爆发点。包含：
- 春季拍卖
- 夏季竞技场天梯对决
- 秋季大丰收排行
- 冬季虚空潮汐世界 BOSS 保卫战

#### 年代循环（长线赛季）
- 每个年代结束后大陆地图重置
- 但玩家保留图鉴、稀有卡牌和个人档案
- 全服的外交决策和战争胜负会由玩家集体谱写成历史档案
- 部分剧情选择会影响下一年代的世界基础设定

### 单机联机无缝切换系统

**核心设计**：所有游戏进度在纯单机和联机模式下完全共享。

- 当处于纯单机状态时，AI 会自动控制外交互动，NPC 扮演你的邻居填充满大陆
- 当处于联机状态时，真实玩家无缝替代这些 NPC

**三种游戏模式**：
1. 离线独自探索
2. 建立 2-8 人的好友私人大陆共享经济圈
3. 加入全服共同大陆参与文明级别的政治和外交层博弈

## 📊 技术选型理由

### 为什么迁移到 Godot？

| 需求 | PixiJS | Godot | 决策 |
|---|---|---|---|
| **联机系统** | 手搓 3 个月 | 内置 ENet/WebSocket | Godot |
| **跨平台发布** | 需封装 (Electron/Capacitor) | 一键导出 | Godot |
| **200+ 灵兽管理** | 纯代码困难 | 场景系统 | Godot |
| **可视化编辑** | 无 | 完整编辑器 | Godot |
| **性能 (2D)** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 相同 |
| **启动速度** | 0.5秒 | 3-8秒 | PixiJS |
| **包体大小** | 500KB | 30-50MB | PixiJS |

**结论**: 长期目标需要联机和跨平台，Godot 是正确选择。

## 🎨 美术风格指南

### 色彩规范

#### 羊皮纸 UI
- 背景: `#f4ecd8` → `#e8dcbf` 渐变（温暖沙色）
- 主边框: `#8b7355` 深棕色，3px double
- 副边框: `rgba(212,175,55,0.5)` 金色，1px solid
- 文字: `#2a2520` 深墨色
- 强调: `#d4af37` 金色

#### 四季色调
- 春 (VER): `Color(1.0, 1.0, 1.0)` 标准
- 夏 (AEST): `Color(1.1, 1.05, 0.95)` 暖黄
- 秋 (AUT): `Color(1.15, 0.95, 0.85)` 橙红
- 冬 (HIE): `Color(0.9, 0.95, 1.1)` 冷蓝

### 字体规范
- **英文标题**: Cormorant Garamond (衬线，优雅)
- **中文正文**: Noto Serif SC (宋体风格)
- **数值**: Cormorant Garamond (斜体)

### 动画曲线
- **入场**: `cubic-bezier(0.2, 0.8, 0.2, 1)` 缓入缓出
- **悬停**: `cubic-bezier(0.65, 0, 0.35, 1)` 弹性
- **淡入淡出**: `cubic-bezier(0.4, 0, 0.2, 1)` 标准
