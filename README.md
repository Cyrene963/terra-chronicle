## v8 新增（跨越恐怖谷 — 三大终极升级）

### 🌊 流体融合水面（Metaball/Threshold 算法）

**之前**：河流是蓝色方块拼接，边缘有明显阶梯锯齿  
**现在**：RenderTexture + Blur + Alpha Threshold = 有机液体

- 将所有水瓦片渲染到离屏纹理 `waterSourceL` → `RenderTexture`
- 应用 `BlurFilter(strength:12, quality:3)` 高斯模糊融合相邻水块
- 应用 `ColorMatrixFilter` alpha 阈值变换：`a_out = clamp(a_in*50 - 22, 0, 1)`  
  → 低于 0.44 的 alpha 被吞噬为 0（透明），高于 0.48 的拉满为 1（满），制造陡峭的"史莱姆边缘"
- 每帧渲染到 `waterRT`，显示在 `waterSprite`

**结果**：瓦片阶梯感彻底消失，河流像真实液体一样圆润流动。这是 2D 游戏做有机流体的行业标准技巧。

### ⚔️ 战斗视觉爆炸 + 精灵生命力

#### 战斗打击感升级（Combat Juice）`src/battle.js`

- **斩击剑气（Slash VFX）**：出攻击牌命中时，敌人身上瞬间划出 2-3 道白色斜斩光带（`180px × 8px` 渐变条，随机角度，screen 混合模式，dissolve fade 350ms）
- **色差畸变（Chromatic Aberration）**：重击时全屏触发 RGB 通道分离滤镜（红色 +3px、青色 -3px 错位），0.2s 衰减回正常，模拟主机游戏的冲击感
- **强化震屏**：伤害 ≥10 时震屏幅度从 15px 提升到 22px，持续 280ms
- 替换原来的简陋黄色光球 → 现在有光弹飞行 + 斜斩 + 色差 + 震屏的完整连击

#### 精灵运动拖尾（Velocity Trails）`src/main.js`

- 水灵兽/火灵兽移动时，每 0.08s 在身后留下一个淡蓝/橙色圆形粒子（5-8px，alpha 0.6）
- 粒子 350ms 内 alpha 衰减到 0 并移除，形成流畅的"残影拖尾"
- 静止时不产生拖尾，只有移动才触发

**结果**：战斗从"数字变化"变成"拳拳到肉"，精灵从"平移纸片"变成"有生命的角色"。

### 🗺️ 肉鸽推图 + 基建升级（Gameplay Loop 完整闭合）

#### 深渊路线图（Roguelike Map）`src/dungeon.js`

**之前**：点传送门 → 直接进战斗，打完掉落，没有选择和策略  
**现在**：Slay-the-Spire 风格节点图

- 3 层深度，每层 3-4 个节点，最终层是 BOSS
- 节点类型：
  - ⚔️ **战斗**（掉落污染种子×1）
  - 👹 **精英**（掉落污染种子×2 + 灵兽灵魂×1，敌人更强）
  - 🔥 **休息**（恢复体力，暂未完整实现）
  - 💀 **BOSS**（掉落污染种子×3 + 灵兽灵魂×2）
- 玩家**选择路径**前进，每个节点都是考验，打完一层解锁下一层
- 路径图用 SVG 线条连接节点，当前可选节点金色边框高亮

#### 农场升级系统（Farm Upgrade）`src/upgrade.js`

**之前**：深渊战利品只能孵化灵兽，没有其他用途  
**现在**：消耗木材 + 灵兽灵魂升级农场基建

- 走近熔炉 → 打开**升级面板**，显示 4 种升级：
  - **工坊 II 级**（木材×8 + 灵魂×2）→ 解锁更强配方，攻防基础值 +40%
  - **工坊 III 级**（木材×16 + 灵魂×4，需 II 级）→ 顶级配方，攻防 +80%，**必出词条**
  - **农田扩建 II**（木材×10 + 灵魂×1）→ 新增 20 块可耕种土地
  - **灵兽栖地**（木材×12 + 灵魂×3）→ 最多同时拥有 4 只灵兽
- 升级持久化到存档，影响后续锻造和农场产能

**完整闭环**：  
Farm（种地产出木材）→ Craft（锻造卡牌）→ Dungeon Map（选路线）→ Battle（战斗获胜）→ Loot（灵兽灵魂 + 污染种子）→ Upgrade（升级工坊/农田）→ 更强卡牌 → 更深地城 → **循环**

玩家现在有了**明确的长线目标**："我要升级工坊到 III 级"、"我要打通 BOSS 拿 3 个灵魂"，不再"玩了 5 分钟后不知道干什么"。

## 操作

- **点击/触摸大地**：寻路移动（自动绕障）· 点树→伐木 · 点耕地→种/收 · 点**深渊之门**→路线图 · 点**孵化阵**→繁育 · 点**熔炉**→升级
- `WASD` 直接移动 · `空格` 就近交互 · `右键` 看地籍 · `1-4` 季节 · `F` 加速时光

## 资产工艺

- `tools/gen_sprites.py` / `gen_heroes.py` / `gen_winter.py` / `gen_round6.py` / `gen_round7.py`：gpt-image-2 `quality:high`
  （实测此代理 size 上限 ~1254²，2K/4K 不被采纳）+ 品红色键（角点采样 + 1px alpha 腐蚀去毛边）+ 无损 PNG-32。
- 换图接口：`src/main.js` 顶部 `ASSETS` 表，`season:[春,夏,秋,冬]` 配四季专属图。
