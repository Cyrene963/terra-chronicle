## 🎯 Gemini 审计指出的 5 大痛点

### 1. 启动黑屏 Bug
**症状**: 进入游戏时右下角黑屏,过一会才正常
**根因**: Canvas DPR 初始化时序问题,视口计算在资源加载后才触发
**解决**: ✅ 已修复

### 2. 操作卡顿
**症状**: 点击移动时感觉"卡卡的"
**根因**: 怀疑 A* 每帧重复计算
**解决**: ✅ 已验证 A* 已优化(每次点击只计算一次),非此问题

### 3. UI 太平庸
**症状**: 现代毛玻璃风格像 SaaS 软件,没有游戏感
**根因**: 冷冰冰的科技审美,不适合奇幻经营游戏
**解决**: ✅ 彻底重构为羊皮纸手账风

### 4. 河面还是假
**症状**: 正方形水块拼斜向河流=生硬阶梯锯齿
**根因**: 缺少对角线过渡贴图
**解决**: ✅ 已生成 4 张对角线资产,待引擎集成

### 5. 核心玩法无特别
**症状**: 种田→打牌太直接干瘪,无化学反应
**根因**: 缺少探索深度和策略层
**解决**: ✅ 炼金大釜配方探索系统上线

### ✅ Task #30: 炼金大釜 + 配方探索系统

**核心改造**: 自动一键锻造 → 手动配方探索

**新增 alchemy.js 模块**:

**配方系统**(6 条隐藏配方):
```

1麦+3木 → 巨盾 (攻6 防28 土系)
2麦+2木 → 平衡刃 (攻16 防14 金系)
4麦+0木 → 生命之粮 (治疗24 光系)
0麦+4木 → 荆棘壁 (攻12 防22 土系)
```

- 配方错误: alert 提示继续试验其他比例

**交互变化**:
- 原"锻造 · 新芽守卫"按钮 → "炼金 · 炼制卡牌"
- 点击打开炼金大釜 UI,完全替代旧 craftCard 逻辑

**结果**: 从"点按钮"→"探索炼金",深度暴增,玩家会为发现配方兴奋

**代码改动**:
